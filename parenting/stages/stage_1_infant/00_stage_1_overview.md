# Stage 1: Infant (0–18 months) — CURRENT

**Status:** ✅ Active — Son is 9 months old (May 2026)

**Develop:** Trust · language foundation · emotional regulation · physical mobility
**How:** Warm repeated interaction — repetition is the wiring mechanism, not a limitation

---

### 🧠 Cognitive
*Build object permanence and cause-effect understanding — through exploratory play, not teaching.*

- [x] Object permanence games habit established — hide toy, let him find it

### 🗣️ Language
*Build language foundation — by treating every babble as a real conversation turn.*

- [x] Daily narration habit established — describe everything during care moments

### ❤️ Social-Emotional
*Build secure attachment — through predictable calm presence when he signals distress.*

- [x] Separation anxiety response established — acknowledge, leave calmly, return reliably

### 🏃 Physical
*Build gross motor development — by removing barriers and letting him lead movement.*

- [x] Floor freedom habit established — crawling and pulling-to-stand encouraged daily
- [ ] Baby-proof the home — outlets, cabinets, stair gates, anchored furniture (window: before walking begins, ~10–12mo)

### 🏥 Health
*Build physical baseline — zero screens, adequate sleep, diverse food before critical windows close.*

- [x] Bedtime ritual established — dim lights → diaper → feed → book → song → same phrase, phones away
- [ ] Food diversity checklist — complete all allergens before 12 months → [checklist](stages/stage_1_infant/food_introduction_checklist.md)
- [ ] Independent sleep — separate surface in same room (window: before 12mo)
- [ ] Feeding shift — solids become primary (window: 12mo)
- [x] Dental — first tooth → same-day brushing + book first dental visit (trigger: first tooth)

### ⚖️ Moral
*Build behavioral boundaries — through warm consistency and rare, physically redirected "No".*

- [x] Consistent warm response posture established

---

**Avoid:** Flashcards, teaching sessions, screens, milestone comparison. Separation anxiety is not a problem to fix.
